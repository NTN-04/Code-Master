Giới thiệu JavaScript

JavaScript là ngôn ngữ lập trình phổ biến nhất thế giới và là một trong ba công nghệ cốt lõi của Web, cùng với HTML và CSS.

JavaScript là ngôn ngữ lập trình nhẹ, thông dịch, hoặc biên dịch just-in-time với các hàm hạng nhất. Mặc dù nó được biết đến nhiều nhất như là ngôn ngữ kịch bản cho các trang Web, nhưng nó cũng được sử dụng trong nhiều môi trường không phải trình duyệt.

Cú pháp cơ bản

JavaScript có cú pháp tương tự như C, Java và Perl, với một số ảnh hưởng từ Python.

// Khai báo biến
let greeting = 'Hello, World!';

// In ra console
console.log(greeting);

// Khai báo hàm
function sayHello(name) {
  return `Hello, ${name}!`;
}

// Gọi hàm
console.log(sayHello('JavaScript'));

Biến và Kiểu dữ liệu

JavaScript có ba từ khóa để khai báo biến: var, let và const.

  • var - Khai báo biến có phạm vi hàm
  • let - Khai báo biến có phạm vi khối
  • const - Khai báo hằng số với phạm vi khối

Các kiểu dữ liệu cơ bản trong JavaScript:

  • String - Văn bản
  • Number - Số nguyên hoặc số thực
  • Boolean - true hoặc false
  • Null - Giá trị đặc biệt nghĩa là "không có gì"
  • Undefined - Biến không được gán giá trị
  • Object - Cấu trúc dữ liệu phức tạp
  • Symbol - Kiểu dữ liệu duy nhất và không thể thay đổi

Hàm

Hàm là một khối mã được thiết kế để thực hiện một tác vụ cụ thể và có thể được gọi nhiều lần.

// Khai báo hàm
function add(a, b) {
  return a + b;
}

// Hàm arrow
const multiply = (a, b) => a * b;

// Hàm như tham số
function calculate(operation, a, b) {
  return operation(a, b);
}

console.log(calculate(add, 5, 3));      // 8
console.log(calculate(multiply, 5, 3)); // 15

Đối tượng

Đối tượng là một tập hợp các thuộc tính, mỗi thuộc tính là một cặp khóa-giá trị.

// Tạo đối tượng
const person = {
  firstName: 'John',
  lastName: 'Doe',
  age: 30,
  fullName: function() {
    return this.firstName + ' ' + this.lastName;
  }
};

// Truy cập thuộc tính
console.log(person.firstName);        // John
console.log(person['lastName']);      // Doe

// Gọi phương thức
console.log(person.fullName());       // John Doe

Mảng

Mảng là một loại đối tượng đặc biệt dùng để lưu trữ nhiều giá trị trong một biến duy nhất.

// Tạo mảng
const fruits = ['Apple', 'Banana', 'Orange'];

// Truy cập phần tử
console.log(fruits[0]); // Apple

// Duyệt mảng
fruits.forEach(fruit => console.log(fruit));

// Thêm phần tử
fruits.push('Mango');

// Phương thức mảng hữu ích
const numbers = [1, 2, 3, 4, 5];
const doubled = numbers.map(num => num * 2);
const sum = numbers.reduce((acc, curr) => acc + curr, 0);

DOM

DOM (Document Object Model) là một giao diện lập trình cho phép JavaScript truy cập và thay đổi cấu trúc và nội dung của một tài liệu HTML.

// Chọn phần tử
const element = document.getElementById('myElement');
const elements = document.querySelectorAll('.myClass');

// Thay đổi nội dung
element.textContent = 'Hello World';
element.innerHTML = 'Hello World';

// Thay đổi style
element.style.color = 'blue';
element.style.backgroundColor = 'yellow';

// Thêm/xóa class
element.classList.add('highlight');
element.classList.remove('hidden');

// Tạo và thêm phần tử mới
const newElement = document.createElement('div');
newElement.textContent = 'Newly created element';
document.body.appendChild(newElement);

Sự kiện

JavaScript cho phép bạn lắng nghe và phản ứng với các sự kiện như nhấp chuột, di chuyển chuột, nhấn phím, v.v.

// Thêm trình xử lý sự kiện
document.getElementById('myButton').addEventListener('click', function(event) {
  alert('Button was clicked!');
});

// Sử dụng arrow function
document.getElementById('myInput').addEventListener('keyup', (e) => {
  console.log('Key pressed:', e.key);
});

// Xóa trình xử lý sự kiện
function handleHover() {
  console.log('Mouse over');
}

const element = document.getElementById('myElement');
element.addEventListener('mouseover', handleHover);
element.removeEventListener('mouseover', handleHover);

JavaScript Bất đồng bộ

JavaScript là ngôn ngữ đơn luồng, nhưng có thể thực hiện các hoạt động bất đồng bộ bằng callbacks, Promises và async/await.

// Callbacks
function fetchData(callback) {
  setTimeout(() => {
    callback('Data fetched');
  }, 2000);
}

fetchData(data => console.log(data));

// Promises
function fetchDataPromise() {
  return new Promise((resolve, reject) => {
    setTimeout(() => {
      resolve('Data fetched with promise');
    }, 2000);
  });
}

fetchDataPromise()
  .then(data => console.log(data))
  .catch(error => console.error(error));

// Async/Await
async function fetchDataAsync() {
  try {
    const data = await fetchDataPromise();
    console.log(data);
  } catch (error) {
    console.error(error);
  }
}

fetchDataAsync();
