Giới thiệu HTML

HTML (HyperText Markup Language) là ngôn ngữ đánh dấu tiêu chuẩn để tạo trang web. HTML mô tả cấu trúc của trang web và bao gồm một loạt các phần tử. Các phần tử HTML cho trình duyệt biết cách hiển thị nội dung.

HTML được phát triển bởi Tim Berners-Lee vào năm 1991. Phiên bản hiện tại là HTML5.

Các thẻ HTML cơ bản

<!DOCTYPE html>
<html>
<head>
  <title>Tiêu đề trang</title>
</head>
<body>
  <h1>Đây là tiêu đề</h1>
  <p>Đây là một đoạn văn.</p>
  <a href="https://example.com">Đây là một liên kết</a>
  <img src="image.jpg" alt="Mô tả hình ảnh">
</body>
</html>
Thẻ Mô tả
<!DOCTYPE html> Xác định loại tài liệu HTML5
<html> Phần tử gốc của trang HTML
<head> Chứa thông tin meta về trang
<title> Tiêu đề trang xuất hiện trên tab trình duyệt
<body> Chứa nội dung trang web hiển thị
<h1> đến <h6> Tiêu đề từ cấp 1 đến cấp 6
<p> Đánh dấu đoạn văn
<a> Tạo liên kết
<img> Hiển thị hình ảnh

Cấu trúc trang HTML

Một trang HTML5 đầy đủ bao gồm các thành phần sau:

<!DOCTYPE html>
<html lang="vi">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Tên trang web</title>
  <meta name="description" content="Mô tả trang web">
  <link rel="stylesheet" href="style.css">
  <script src="script.js" defer></script>
</head>
<body>
  <header>
    <nav>
      <!-- Thanh điều hướng -->
    </nav>
  </header>
  
  <main>
    <section>
      <h1>Tiêu đề chính</h1>
      <article>
        <!-- Nội dung bài viết -->
      </article>
    </section>
    
    <aside>
      <!-- Thanh bên -->
    </aside>
  </main>
  
  <footer>
    <!-- Chân trang -->
  </footer>
</body>
</html>

Biểu mẫu HTML

HTML cung cấp nhiều phần tử để tạo biểu mẫu tương tác với người dùng: 

<form action="/submit" method="post">
  <fieldset>
    <legend>Thông tin cá nhân</legend>
    
    <label for="name">Họ tên:</label>
    <input type="text" id="name" name="name" required>
    
    <label for="email">Email:</label>
    <input type="email" id="email" name="email" required>
    
    <label for="password">Mật khẩu:</label>
    <input type="password" id="password" name="password" required>
    
    <label for="gender">Giới tính:</label>
    <select id="gender" name="gender">
      <option value="male">Nam</option>
      <option value="female">Nữ</option>
      <option value="other">Khác</option>
    </select>
    
    <label>
      <input type="checkbox" name="subscribe" value="yes">
      Đăng ký nhận tin
    </label>
    
    <button type="submit">Gửi</button>
    <button type="reset">Xóa</button>
  </fieldset>
</form>

HTML ngữ nghĩa

HTML5 giới thiệu nhiều phần tử ngữ nghĩa giúp định nghĩa rõ ràng các phần của trang web:

Thẻ Mô tả
<header> Phần đầu của trang hoặc phần
<nav> Phần điều hướng
<main> Nội dung chính của trang
<article> Nội dung độc lập, có thể phân phối riêng
<section> Phần của trang có chủ đề liên quan
<aside> Nội dung liên quan đến nội dung xung quanh
<footer> Phần chân của trang hoặc phần
<figure> và <figcaption> Hình ảnh hoặc minh họa và chú thích

Phương tiện trong HTML

HTML5 cung cấp các phần tử tích hợp để nhúng âm thanh, video và các phương tiện tương tác khác: 

<!-- Video -->
<video width="320" height="240" controls>
  <source src="movie.mp4" type="video/mp4">
  <source src="movie.ogg" type="video/ogg">
  Trình duyệt của bạn không hỗ trợ thẻ video.
</video>

<!-- Audio -->
<audio controls>
  <source src="audio.mp3" type="audio/mpeg">
  <source src="audio.ogg" type="audio/ogg">
  Trình duyệt của bạn không hỗ trợ thẻ audio.
</audio>

<!-- Canvas -->
<canvas id="myCanvas" width="200" height="100">
  Trình duyệt của bạn không hỗ trợ thẻ canvas.
</canvas>

API HTML5

HTML5 đi kèm với nhiều API JavaScript để tạo ứng dụng web phong phú:

  • Geolocation API - Xác định vị trí người dùng
  • Drag and Drop API - Kéo và thả các phần tử
  • Local Storage API - Lưu trữ dữ liệu cục bộ
  • Session Storage API - Lưu trữ dữ liệu phiên
  • Web Workers API - JavaScript đa luồng
  • WebSocket API - Giao tiếp hai chiều
  • Canvas API - Vẽ đồ họa 2D
  • History API - Thao tác lịch sử trình duyệt
