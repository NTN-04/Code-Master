Giới thiệu HTML

HTML (HyperText Markup Language) là ngôn ngữ đánh dấu tiêu chuẩn để tạo trang web. HTML mô tả cấu trúc của trang web và bao gồm một loạt các phần tử. Các phần tử HTML cho trình duyệt biết cách hiển thị nội dung.

HTML được phát triển bởi Tim Berners-Lee vào năm 1991. Phiên bản hiện tại là HTML5.

Cấu trúc cơ bản của một tài liệu HTML


<!DOCTYPE html>
<html>
  <head>
    <meta charset="UTF-8">
    <title>Tiêu đề trang</title>
  </head>
  <body>
    <h1>Chào mừng đến với HTML!</h1>
    <p>Đây là một đoạn văn bản.</p>
  </body>
</html>
Thành phần Chức năng
<!DOCTYPE html> Khai báo loại tài liệu HTML
<html> Bao quanh toàn bộ nội dung HTML
<head> Chứa thông tin meta như tiêu đề, charset
<body> Chứa nội dung hiển thị trên trang web

Một số thẻ HTML thông dụng

Thẻ Công dụng
<h1> đến <h6> Tiêu đề, từ lớn nhất đến nhỏ nhất
<p> Đoạn văn bản
<a> Liên kết (hyperlink)
<img> Chèn hình ảnh
<ul> / <ol> Danh sách (unordered & ordered)
<table> Bảng dữ liệu
<form> Biểu mẫu nhập liệu

Ví dụ đơn giản


<h2>Giới thiệu bản thân</h2>
<p>Tôi tên là nguyen và đang học về thiết kế web.</p>
<a href="https://example.com">Trang cá nhân</a>

Các thẻ HTML cơ bản

<!DOCTYPE html>
<html>
<head>
  <title>Tiêu đề trang</title>
</head>
<body>
  <h1>Đây là tiêu đề</h1>
  <p>Đây là một đoạn văn.</p>
  <a href="https://example.com">Đây là một liên kết</a>
  <img src="image.jpg" alt="Mô tả hình ảnh">

</html>

Dưới đây là danh sách các thẻ phổ biến và chức năng tương ứng:

Thẻ Chức năng Ví dụ
<!DOCTYPE> Khai báo kiểu tài liệu HTML <!DOCTYPE html>
<html> Bao quanh toàn bộ nội dung <html>...</html>

<a> Liên kết (hyperlink) <a href="https://example.com">Click</a>

<br> Xuống dòng Dòng thứ nhất<br>Dòng thứ hai

Cấu trúc trang HTML

Một trang HTML5 đầy đủ bao gồm các thành phần sau:

<!DOCTYPE html>
<html lang="vi">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Tên trang web</title>
  <meta name="description" content="Mô tả trang web">
  <link rel="stylesheet" href="style.css">
  <script src="script.js" defer></script>
</head>
<body>
  <header>
    <nav>
      <!-- Thanh điều hướng -->
    </nav>
  </header>
  
  <main>
    <section>
      <h1>Tiêu đề chính</h1>
      <article>
        <!-- Nội dung bài viết -->
      </article>
    </section>
    
    <aside>
      <!-- Thanh bên -->
    </aside>
  </main>
  
  <footer>
    <!-- Chân trang -->
  </footer>
</body>
</html>

Biểu mẫu HTML (Forms)

Biểu mẫu HTML (Forms) là tập hợp các trường nhập liệu (input fields), nút bấm và các phần tử khác mà người dùng có thể dùng để gửi thông tin đến máy chủ hoặc xử lý bằng JavaScript.
Tất cả nằm bên trong cặp thẻ <form>...</form>

Các thành phần cơ bản của biểu mẫu

1. Thẻ <form>


<form action="/xuly" method="POST">
  <!-- các trường dữ liệu -->
</form>
Thuộc tính Ý nghĩa
action URL nhận dữ liệu gửi đi
method Phương thức gửi: GET hoặc POST
target Nơi mở kết quả (_blank, _self)

2. Các trường nhập liệu

Thẻ Chức năng Ví dụ
<input type="text"> Nhập văn bản ngắn Tên, email,...
<input type="password"> Nhập mật khẩu Mật khẩu
<input type="email"> Nhập địa chỉ email Email
<input type="number"> Nhập số Tuổi, số lượng
<textarea> Nhập văn bản dài Mô tả, nội dung
<select> + <option> Menu chọn một hoặc nhiều mục Quốc gia, ngành nghề,...
<input type="radio"> Chọn 1 trong nhiều tùy chọn Giới tính, trạng thái,...
<input type="checkbox"> Chọn nhiều tùy chọn Sở thích
<input type="submit"> Nút gửi dữ liệu "Gửi"
<input type="reset"> Xóa toàn bộ dữ liệu đã nhập "Xóa"

Ví dụ biểu mẫu đơn giản


<form action="/guithongtin" method="POST">
  <label for="ten">Họ tên:</label>
  <input type="text" id="ten" name="ten"><br><br>

  <label for="email">Email:</label>
  <input type="email" id="email" name="email"><br><br>

  <label for="gioitinh">Giới tính:</label><br>
  <input type="radio" id="nam" name="gioitinh" value="nam"> Nam
  <input type="radio" id="nu" name="gioitinh" value="nu"> Nữ<br><br>

  <label for="ghichu">Ghi chú:</label><br>
  <textarea name="ghichu" rows="4" cols="30"></textarea><br><br>

  <input type="submit" value="Gửi">
</form>
Kết quả: Người dùng điền tên, email, chọn giới tính, ghi chú rồi nhấn “Gửi”.

Mẹo và lưu ý

  • Luôn thêm thuộc tính name để dữ liệu gửi đi được nhận diện đúng.
  • Dùng <label> để gắn nhãn cho input giúp tăng khả năng truy cập (accessibility).
  • Kết hợp JavaScript để kiểm tra dữ liệu trước khi gửi (form validation).
  • Có thể dùng required để bắt buộc điền thông tin: <input type="email" required>

HTML ngữ nghĩa

HTML5 giới thiệu nhiều phần tử ngữ nghĩa giúp định nghĩa rõ ràng các phần của trang web:

Thẻ Mô tả
<header> Phần đầu của trang hoặc phần
<nav> Phần điều hướng
<main> Nội dung chính của trang
<article> Nội dung độc lập, có thể phân phối riêng
<section> Phần của trang có chủ đề liên quan
<aside> Nội dung liên quan đến nội dung xung quanh
<footer> Phần chân của trang hoặc phần
<figure> và <figcaption> Hình ảnh hoặc minh họa và chú thích

Phương tiện trong HTML

HTML5 cung cấp các phần tử tích hợp để nhúng âm thanh, video và các phương tiện tương tác khác: 

<!-- Video -->
<video width="320" height="240" controls>
  <source src="movie.mp4" type="video/mp4">
  <source src="movie.ogg" type="video/ogg">
  Trình duyệt của bạn không hỗ trợ thẻ video.
</video>

<!-- Audio -->
<audio controls>
  <source src="audio.mp3" type="audio/mpeg">
  <source src="audio.ogg" type="audio/ogg">
  Trình duyệt của bạn không hỗ trợ thẻ audio.
</audio>

<!-- Canvas -->
<canvas id="myCanvas" width="200" height="100">
  Trình duyệt của bạn không hỗ trợ thẻ canvas.
</canvas>

API HTML5

Trong ngữ cảnh HTML5, “API” (Application Programming Interface) là giao diện lập trình cho phép trang web làm được những tác vụ trước đây chỉ dành cho ứng dụng phần mềm. Ví dụ như: lưu dữ liệu cục bộ, theo dõi vị trí người dùng, vẽ đồ họa 2D, ghi âm video, v.v.

Một số HTML5 API phổ biến và mạnh mẽ

Tên API Mô tả chức năng chính Ví dụ ứng dụng thực tế
Canvas API Vẽ đồ họa 2D trực tiếp trên trình duyệt Game 2D, đồ thị, trình chỉnh ảnh đơn giản
Geolocation API Truy cập vị trí của người dùng qua GPS/WiFi Bản đồ, tìm địa điểm gần nhất
Web Storage API Lưu trữ dữ liệu trên trình duyệt (local/sessionStorage) Giỏ hàng, lưu trạng thái người dùng
Media API Truy cập webcam, microphone của người dùng Gọi video, quay video từ trình duyệt
Drag and Drop API Kéo thả phần tử từ nơi này sang nơi khác Quản lý file, sắp xếp ảnh, upload dễ dàng
Web Workers API Chạy tác vụ nền mà không làm chậm giao diện chính Tính toán phức tạp, xử lý dữ liệu lớn
Notifications API Hiển thị thông báo cho người dùng Cảnh báo, tin mới, nhắc việc
FullScreen API Chuyển website sang chế độ toàn màn hình Trình chiếu, video, game
Speech Recognition Nhận dạng giọng nói để điều khiển website Tìm kiếm bằng giọng nói, điều khiển bằng lời
Battery Status API Biết mức pin thiết bị người dùng (nay ít được hỗ trợ) Tối ưu giao diện với pin thấp
