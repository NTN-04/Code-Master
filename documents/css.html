Giới thiệu CSS

CSS (Cascading Style Sheets) là ngôn ngữ được sử dụng để tạo kiểu cho trang web HTML. CSS mô tả cách các phần tử HTML nên được hiển thị trên màn hình, trên giấy, hoặc trong các phương tiện khác.

CSS tiết kiệm rất nhiều công việc. Nó có thể kiểm soát bố cục của nhiều trang web cùng lúc. 

/* Đây là một bình luận CSS */
body {
  background-color: lightblue;
}

h1 {
  color: navy;
  margin-left: 20px;
}

Bộ chọn CSS

Bộ chọn CSS được sử dụng để "tìm" (hoặc chọn) các phần tử HTML bạn muốn tạo kiểu.

Chúng ta có thể chia các bộ chọn CSS thành năm danh mục:

  • Bộ chọn đơn giản (chọn phần tử dựa trên tên, id, lớp)
  • Bộ chọn kết hợp (chọn phần tử dựa trên mối quan hệ cụ thể giữa chúng)
  • Bộ chọn lớp giả (chọn phần tử dựa trên một trạng thái nhất định)
  • Bộ chọn phần tử giả (chọn và tạo kiểu cho một phần của phần tử)
  • Bộ chọn thuộc tính (chọn phần tử dựa trên thuộc tính hoặc giá trị thuộc tính)

Mô hình hộp

Trong CSS, mô hình hộp là một hộp bao quanh mọi phần tử HTML. Nó bao gồm: lề (margins), đường viền (borders), phần đệm (padding) và nội dung thực tế.

Mô hình hộp cho phép chúng ta thêm đường viền xung quanh phần tử và xác định không gian xung quanh phần tử. 

div {
  width: 300px;
  padding: 25px;
  border: 5px solid navy;
  margin: 25px;
}

Bố cục CSS

CSS cung cấp nhiều thuộc tính khác nhau để kiểm soát cách các phần tử được bố trí trong trang web:

  • display - Xác định loại hộp hiển thị của phần tử
  • position - Xác định phương pháp định vị được sử dụng cho phần tử
  • float - Xác định cách phần tử nên float
  • clear - Xác định bên nào của phần tử không được phép nổi

Flexbox

Mô-đun Bố cục Flexbox làm cho việc thiết kế cấu trúc bố cục linh hoạt dễ dàng hơn mà không cần sử dụng float hoặc định vị.

Trước khi Flexbox Layout module, có bốn chế độ bố cục: Block (cho phần), Inline (cho văn bản), Table, và Positioned. 

.flex-container {
  display: flex;
  flex-direction: row;
  justify-content: center;
  align-items: center;
}

CSS Grid

CSS Grid Layout cung cấp một hệ thống bố cục dựa trên lưới, với hàng và cột, giúp dễ dàng thiết kế các trang web mà không cần sử dụng float và định vị.

CSS Grid là công nghệ mạnh mẽ nhất có sẵn trong CSS để tạo bố cục. 

.grid-container {
  display: grid;
  grid-template-columns: auto auto auto;
  gap: 10px;
  background-color: #2196F3;
  padding: 10px;
}

Thiết kế đáp ứng

Thiết kế đáp ứng là về việc sử dụng CSS để làm cho một trang web trông đẹp trên tất cả các thiết bị.

Thiết kế đáp ứng không phải là một chương trình hoặc plugin JavaScript. Nó chỉ sử dụng HTML và CSS. 

@media only screen and (max-width: 600px) {
  body {
    background-color: lightblue;
  }
}
